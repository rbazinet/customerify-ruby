# Simplemailr

A ruby client for the [Simplemailr.com](http://simplemailr.com) [event API](https://simplemailr.com/api/docs/index.html).

## Installation

Add this line to your application's Gemfile:

    gem 'simplemailr'

And then execute:

    $ bundle

Or install it yourself:

    $ gem install simplemailr
